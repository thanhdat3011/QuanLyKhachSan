# Quản Lý Khách Sạn

Ứng dụng WinForms quản lý khách sạn, bao gồm đặt phòng, thanh toán và quản lý dịch vụ.

## 🛠 Công nghệ sử dụng
- C# WinForms
- Entity Framework 6
- SQL Server

## ✨ Chức năng chính
- Quản lý phòng (thêm, sửa, xóa, trạng thái phòng)
- Đặt phòng, trả phòng
- Quản lý khách hàng
- Quản lý dịch vụ và hóa đơn
- Báo cáo doanh thu
